## 아이스브레이킹 규칙 (PRD 14.1)
- 반드시 3개 생성
- 세 멘트는 하나의 자연스러운 대화 흐름으로 연결되어야 함
- 첫 번째: 따뜻한 감사 인사 + 긴장 완화 (예: "오늘 바쁘신 중에 시간 내주셔서 정말 감사해요. 편하게 말씀해 주세요.")
- 두 번째: 페르소나의 현재 상황에 공감하며 친밀감 형성 (첫 번째 멘트에서 자연스럽게 이어지는 말투로)
- 세 번째: 인터뷰 목적을 부드럽게 안내하며 심리적 안전감 제공 (두 번째 멘트에서 자연스럽게 이어지는 말투로)
- 세 멘트 전체가 읽혔을 때 하나의 짧은 오프닝 모놀로그처럼 느껴져야 함
- 각 멘트는 2~3문장 이내로 간결하게
- 절대 금지: 솔루션/제품 언급, 형식적인 인사말 반복

예시 (좋은 예):
- "안녕하세요, 오늘 바쁘신 중에 시간 내주셔서 정말 감사해요. 편하게 말씀해 주시면 돼요."
- "요즘 [페르소나 관련 상황]으로 정말 바쁘실 것 같아요. 저도 주변에서 비슷한 이야기를 많이 듣거든요."
- "오늘은 정답이 없는 대화예요. 솔직하게 말씀해 주실수록 정말 도움이 돼요. 시작해볼까요?"

## 각 질문 필수 구성 요소 (PRD 5.2, 8.3)

각 질문마다 반드시 아래 3가지를 포함하세요:

1. 질문 본문 (오픈엔디드, 과거 행동/경험/감정 중심)
2. 왜 이 질문을 하는가? (1~3문장, 린 고객개발 원칙 연결)
3. 기대 답변 패턴 (강한 Pain 신호 vs 약한 신호 예시)
4. 후속 질문 2~3개 (공감 리액션 포함, 예: "힘들었겠어요, 더 자세히 들려주실 수 있을까요?")
